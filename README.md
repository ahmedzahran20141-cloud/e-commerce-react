# 🛍️ E-Commerce & Admin Management Dashboard

A full-featured, professional E-Commerce Web Application built with **React.js**, **Bootstrap**, and **RESTful API** integration. Designed for seamless customer shopping and complete administrative product control.

---

## ✨ Features

### 🛒 Customer Features
* **Product Catalog:** Browse products with real-time category filtering and dynamic sorting by price.
* **Pagination:** Smooth pagination for quick browsing through large product catalogs.
* **Product Details:** Dedicated page showing full product specifications with image fallback handling.
* **Shopping Cart System:** Reactive shopping cart with live item counter in the navigation bar.
* **Order Management & Checkout:** Interactive modal for choosing payment methods (Cash / Card) with instant order status updates and history logs.

### 🔐 Admin Dashboard
* **Protected Routes:** Admin-only pages secured via custom Context Authentication.
* **CRUD Operations:** Complete functionality to Add, View, Edit, and Delete store products using SweetAlert2 confirmations.
* **Form Validation:** Input checks for valid price ranges and required media links.
* **Image Fallback System:** Automatic detection and replacement of broken image URLs using custom SVG/Placeholder components.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, React Router v6, Context API
* **Styling & UI:** Bootstrap 5, SweetAlert2, React Toastify
* **Security & Auth:** Base64 Authentication Masking, Route Guarding
* **Backend Integration:** Mock REST API via `json-server`
